基本数据类型三

字典

获取值

第一种方法： 通过key值去取value值

第二种方法 get –> D.get(k[,d]) -> D[k] if k in D, else d. d defaults to None.

第三种方法 setdefault –> D.setdefault(k[,d]) -> D.get(k,d), also set D[k]=d if k not in D

解释: 通过给定的key查找对应的value值, 如果给定的key==在字典中==,返回对应的==vaule==, 如果不在,返回默认值,同时,在字典中添加了key: 默认值的键对(默认为None,如果设置了d, 则使用d)

增 （两种）

第一种， 通过 key， key有则改， 无则谓

第二种： setdefault([k].d) # – > 获取, 不会改变

key值有则获取， 无则新增

删除 –> 三种方法 pop popitem clear

集合

一个包含唯一元素的可变和无序的集合数据类型

变量初始化， 同列表类似， 元素可以是任何数据类型（排除列表）

remove

pop

discrad

如果有， 直接删除， 如果没有， 不做任何的操作
